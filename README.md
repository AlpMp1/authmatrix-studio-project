# AuthMatrix Studio

AuthMatrix Studio 是一个 AI 授权回归检测与策略修复工作台，面向 SaaS 产品和后端 API。

## 项目能力
- 读取路由处理器、OpenAPI、RBAC/ABAC 策略、权限矩阵、审计日志、测试用例和 PR diff
- 发现越权风险、租户隔离缺口、角色继承错误和策略覆盖不足
- 生成路由策略绑定、租户边界检查、负向回归测试和审计日志验证
- 输出 PR 级权限变更报告、验证矩阵和剩余风险说明

## 部署方式
这是一个静态 HTML demo，可以直接部署到 GitHub Pages、Netlify、Vercel 或任意静态托管平台。
