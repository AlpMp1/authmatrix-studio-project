AuthMatrix Studio 是一个面向 SaaS 产品和后端 API 的 AI 授权回归检测与策略修复工作台。它会读取路由处理器、OpenAPI、RBAC/ABAC 策略、权限矩阵、审计日志、测试用例和 PR diff，自动发现越权风险、租户隔离缺口、角色继承错误和策略覆盖不足，并生成可审查的修复 patch 与回归测试。

Designed for authorization changes that are hard to review manually.

AuthMatrix 不只是扫描是否调用了 auth 函数，而是把路由、策略、角色继承、租户边界、审计日志和测试用例串成权限图，帮助团队发现真实业务路径中的授权回归。
